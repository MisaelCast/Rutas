# Pokédex

Aplicación web construida con Vue 3 que consume la PokéAPI.

## Tecnologías

- Vue 3
- Vue Router
- Axios
- Tailwind CSS
- Vite

## Funciones

- Lista de pokémons con paginación
- Vista de detalle con stats
- Navegación entre páginas

## Instalación

```bash
npm install
npm run dev
```

## API

Datos obtenidos de [PokéAPI](https://pokeapi.co)
